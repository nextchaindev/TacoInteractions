## Quickstart
1. Set up Redis/KeyDB
2. Set up a PostgreSQL Database
3. Create a Discord Bot Application
4. `yarn && npx prisma generate`
5. Copy `.env.example` into `.env` and fill in the variables
6. `yarn start`
